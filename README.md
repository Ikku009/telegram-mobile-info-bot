# 📱 Telegram Mobile Info Bot

A simple and powerful Telegram bot that retrieves detailed information about phone numbers worldwide.

**Created by: @Joh4n404**

## Features

✅ **Phone Number Validation** - Validates international phone numbers  
✅ **Country Detection** - Identifies country and region  
✅ **Carrier Information** - Shows network provider details  
✅ **Number Type Detection** - Identifies if number is mobile, fixed line, toll-free, etc.  
✅ **Easy to Use** - Simple commands and direct message handling  
✅ **International Support** - Works with phone numbers from any country
